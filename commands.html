Git Commands

  • git init: to initialize the git in a project.
  • git clone: creates a copy of the existing git repository.
  • git commit: takes the changes made upto te current stage and commits them to the project history.
  • git log: Lets you check the previous commits implemented in the project.
  • git status: Tells you the state of the files in the project.
  • git pull: It helps in fetching the files from github onto the local repository.
  • git push: It helps in saving the local repository onto github.
  • git reset: Helps you to undo the changes made in the repository upto a specific stage.
  • git branch: Its used to create a test repository.
  • git checkout: It is used to navigate between different branches.
  • git merge: Its used to combine the data contained branches.
